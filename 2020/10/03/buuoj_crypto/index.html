RSA5

百度百科:最小公倍数性质

首先先得知道这个公式：(a, b) * [a, b] = a * b

  • (a, b)表示a,b的最大公因数

  • [a, b]表示a,b的最小公倍数

    由rsa公式有:n = p * q

​ phi = (p - 1) * (q - 1)

所以n 和 phi 应该是两个大小长度差不多的数

计算 n 与 gift 的倍数差值

以这个值为上界爆破得到phi

根据rsa基本原理有公式: c = me mod d

​ 但是这个题的坑就在于 e 并不是素数，e/2才是

公式改写为: c = (m2)e/2 mod d

​ 所以我们计算出来的应该是m2，开方得到m
